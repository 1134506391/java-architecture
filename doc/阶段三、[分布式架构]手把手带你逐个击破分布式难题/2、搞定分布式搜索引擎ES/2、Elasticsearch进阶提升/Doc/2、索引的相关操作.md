# 索引的基本操作

## 1、集群健康

https://www.elastic.co/guide/cn/elasticsearch/guide/current/_cluster_health.html#_cluster_health

```
GET     /_cluster/health
```

## 2、创建索引

```
PUT     /index_test
{
    "settings": {
        "index": {
            "number_of_shards": "2",
            "number_of_replicas": "0"
        }
    }
}
```

## 3、查看索引

```
GET     _cat/indices?v
```

## 4、删除索引

```
DELETE      /index_test
```
